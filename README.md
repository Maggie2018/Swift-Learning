"Learning by doing" is the best way to learn swift & iOS development.
# Swift Learning Path
- 了解开发工具
- 认真学习并练习Swift基本语法
- 理解界面和代码的交互
- 理解项目框架
- 理解设计模式，如MVC
- 了解app生命周期
- 根据想开发的app类型进行针对性学习
